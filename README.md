# Diprion_pini_genome
Assembly and annotation of the genome of the Common pine sawfly (Diprion pini)

Based on PacBio HiFi reads, Oxford Nanopore MionION long reads, and 10x Genomics linked reads, we present the highly continuous, near chromosome-scale assembly, structural and functional annotation of the genome of the common pine sawfly (Diprion pini). This genome constitutes a resource for improved pest management for this important defoliator of pine forests and offers opportunities for resolving longstanding questions about Hymenoptera evolution. 

Here we provide the code and necessary files to reproduce the analyses. The annotation files are also available here.
